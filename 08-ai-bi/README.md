# 08. AI/BI (비즈니스 인텔리전스)

> Databricks의 AI/BI 기능을 활용한 데이터 시각화와 비즈니스 분석을 학습합니다.

## 학습 목표
- AI/BI 대시보드(Lakeview)를 활용한 데이터 시각화
- Genie를 통한 자연어 기반 데이터 탐색
- 대시보드 공유와 스케줄링
- 알림(Alerts) 설정을 통한 데이터 모니터링
