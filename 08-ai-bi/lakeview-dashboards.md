# AI/BI 대시보드

## 대시보드 생성

1. 좌측 메뉴 **Dashboards** → **Create Dashboard**
2. 대시보드 이름 입력
3. **Dataset** 추가: SQL 쿼리로 데이터 정의
4. **위젯** 추가: 차트, 표, 카운터, 텍스트 등
5. 필터 추가: 날짜 범위, 카테고리 등

---

## 위젯 유형

| 위젯 | 용도 |
|------|------|
| **Bar Chart** | 카테고리별 비교 |
| **Line Chart** | 시계열 추이 |
| **Pie / Donut** | 비율 비교 |
| **Counter** | 단일 KPI 수치 표시 |
| **Table** | 상세 데이터 테이블 |
| **Scatter** | 두 변수 간 관계 |
| **Pivot Table** | 크로스탭 분석 |

---

## 대시보드 공유

| 방법 | 설명 |
|------|------|
| **Workspace 내 공유** | 사용자/그룹에게 View/Edit 권한 부여 |
| **스케줄 이메일** | 정해진 시간에 스냅샷을 이메일로 전송 |
| **PDF 다운로드** | 현재 상태를 PDF로 저장 |
| **임베딩** | iframe으로 외부 사이트에 삽입 |
