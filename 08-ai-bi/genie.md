# Genie — 자연어 데이터 분석

## Genie란?

> 💡 **Genie**는 자연어로 데이터에 질문하면 **SQL 쿼리를 자동으로 생성**하여 결과를 보여주는 AI 기반 분석 도구입니다.

"이번 달 서울 지역 매출이 얼마야?"라고 물으면, Genie가 적절한 SQL을 작성하고 실행하여 답변을 제공합니다.

---

## Genie Space 구성

| 구성 요소 | 설명 |
|-----------|------|
| **테이블 선택** | Genie가 참조할 테이블을 지정합니다 |
| **지시사항** | "금액은 원 단위입니다", "지역은 city 컬럼입니다" 같은 맥락을 제공합니다 |
| **예시 질문** | 자주 물을 수 있는 질문의 예시를 등록합니다 |
| **인증된 답변** | 자주 묻는 질문에 대해 검증된 SQL을 미리 등록합니다 |

> 🆕 **Genie Code (GA)**: 기존 Databricks Assistant가 확장되어, 멀티 스텝 데이터 작업을 자율적으로 수행할 수 있게 되었습니다.
