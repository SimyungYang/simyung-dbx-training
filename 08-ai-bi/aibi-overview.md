# AI/BI 개요

## Databricks AI/BI란?

> 💡 **AI/BI**는 Databricks의 비즈니스 인텔리전스 솔루션으로, **AI/BI Dashboard (Lakeview)**와 **Genie**를 핵심 구성 요소로 합니다. SQL 쿼리 결과를 시각화하고, 자연어로 데이터에 질문할 수 있습니다.

| 구성 요소 | 역할 |
|-----------|------|
| **AI/BI Dashboard** | SQL 기반 데이터 시각화 대시보드입니다 |
| **Genie** | 자연어로 데이터에 질문하면 SQL을 자동 생성하여 답변합니다 |
| **Alerts** | 쿼리 결과가 조건을 만족하면 자동으로 알림을 보냅니다 |
