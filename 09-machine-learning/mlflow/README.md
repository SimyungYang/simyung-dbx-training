# 09-1. MLflow

> 오픈소스 ML 플랫폼 MLflow를 활용한 실험 추적, 모델 관리, GenAI 트레이싱을 학습합니다.

## 학습 목표
- MLflow의 핵심 컴포넌트(Tracking, Models, Registry) 이해
- 실험(Experiment)과 런(Run) 관리
- 모델 로깅, 등록, 버전 관리
- GenAI 앱에 대한 트레이싱(Tracing) 설정
