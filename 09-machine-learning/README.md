# 09. 머신러닝 (Machine Learning)

> Databricks 위에서 ML 모델을 개발, 추적, 배포하는 전체 과정을 학습합니다.

## 학습 목표
- MLflow를 활용한 실험 추적과 모델 관리
- Unity Catalog 기반 모델 레지스트리 사용법
- Feature Engineering과 Feature Store 활용
- Model Serving을 통한 모델 배포와 실시간 추론
